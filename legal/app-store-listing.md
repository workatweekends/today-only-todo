## 説明文（4000文字以内）— こちらをコピペ

```
「昨日の未完了、今日も残ってる…」
そんな罪悪感を抱えたことはありませんか？

今日だけToDoは、翌日になるとタスクがすべて消える、新しいタイプのToDoアプリです。持ち越しなし、プレッシャーなし。毎朝まっさらな状態から、今日やることだけに集中できます。


■ 今日だけToDoとは

多くのToDoアプリは、未完了のタスクがどんどん溜まっていきます。リストが長くなるほど、開くのがつらくなる。完璧にこなせなかった自分を責めてしまう。そんな経験はありませんか？

今日だけToDoは、その悩みを根本から見直すアプリです。

設定した時刻（デフォルトは午前0時）を過ぎると、すべてのタスクは自動的にリセットされます。昨日できなかったことは、昨日で終わり。今日は今日の分だけ、ゼロから始められます。


■ 主な機能

【タスク管理】
・テキストを入力してEnterで、すぐにタスクを追加
・タップで完了チェック。取り消し線アニメーションつき
・左スワイプで削除

【進捗の可視化】
・「2 / 5 完了」のように、今日の進捗をひと目で確認
・プログレスバーで達成感を表示
・全タスク完了時には、お祝いの紙吹雪アニメーション

【今日を意識するデザイン】
・「あと○時間」で今日の残り時間をカウントダウン
・シンプルで余白のある、青と白を基調としたUI
・ダークモード対応（システム設定・手動切り替え）

【カスタマイズ】
・リセット時刻を自由に変更（例：午前3時にリセットなど）
・ライト / ダーク / システム連動のテーマ設定


■ こんな方におすすめ

・ToDoリストが際限なく増えて、ストレスになっている
・「昨日やれなかったこと」を引きずりたくない
・完璧主義で、かえって手が止まってしまう
・毎日を気持ちよくリセットして、軽く始めたい
・機能が多すぎない、シンプルなアプリが好き
・学生、社会人、主婦・主夫など、日々のタスク管理に悩むすべての方


■ 使い方

1. アプリを開く
2. 「今日やること」を入力してタスクを追加
3. できたらタップして完了
4. 翌日になると、すべてのタスクが自動でリセット

それだけです。アカウント登録は不要。難しい設定もありません。


■ データの取り扱いについて

・タスクデータはお使いの端末内にのみ保存されます
・当方のサーバーにデータを送信することはありません
・アカウント登録やログインは不要です
・アプリを削除すると、すべてのデータも削除されます

詳しくはプライバシーポリシーをご確認ください。


■ 広告について

本アプリは無料でご利用いただけます。運営のため、画面下部にバナー広告を表示しています（Google AdMob）。


■ お問い合わせ

ご意見・ご要望・不具合の報告は、以下までお気軽にご連絡ください。
workatweekends.m@gmail.com


毎日を、もっと軽く。
今日だけToDoで、ゼロから始めましょう。
```

## スクリーンショット撮影ガイド

### 必要なサイズ（iPhone）

App Store Connect は **6.7インチ**（iPhone 15 Pro Max 等）のスクリーンショットが必須です。
シミュレータで **iPhone 17 Pro Max** を使えば OK です。

| # | 画面 | 撮る内容 |
|---|------|---------|
| 1 | メイン（タスクあり） | タスク3〜5件、進捗バー表示 |
| 2 | メイン（空状態） | 「今日やることを追加しよう」 |
| 3 | タスク完了 | 1件完了＋取り消し線の状態 |
| 4 | 全完了 | 紙吹雪アニメーションの瞬間 |
| 5 | 設定画面 | リセット時刻・ダークモード |

### 撮影コマンド（シミュレータ起動中）

```bash
# 保存先フォルダを作成
mkdir -p ~/Desktop/ToDo-screenshots

# スクリーンショット撮影（番号を変えて5枚）
xcrun simctl io booted screenshot ~/Desktop/ToDo-screenshots/01-main.png
xcrun simctl io booted screenshot ~/Desktop/ToDo-screenshots/02-empty.png
xcrun simctl io booted screenshot ~/Desktop/ToDo-screenshots/03-complete.png
xcrun simctl io booted screenshot ~/Desktop/ToDo-screenshots/04-confetti.png
xcrun simctl io booted screenshot ~/Desktop/ToDo-screenshots/05-settings.png
```

Mac のショートカット **⌘ + S**（シミュレータがアクティブ時）でも撮影できます。
